# MG-0022: Monster Party UI/UX 스펙

> **게임명**: Monster Party (몬스터 파티)
> **장르**: 파티 미니게임
> **카테고리**: Year 2 Core
> **문서 버전**: 1.0.0

---

## 1. 화면 설정

| 항목 | 값 |
|------|-----|
| **방향** | Portrait (세로) |
| **기준 해상도** | 1080 × 1920 |
| **지원 비율** | 16:9 ~ 21:9 |
| **타겟 FPS** | 60fps |

---

## 2. Safe Area

| 영역 | 값 | 용도 |
|------|-----|------|
| 상단 | 44-62dp | 점수, 라운드 |
| 하단 | 34-48dp | 조작 버튼 |
| 좌우 | 16dp | 콘텐츠 여백 |
