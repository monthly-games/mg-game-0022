## 2. 코어 루프 (Core Loop)

```
┌─────────────────────────────────────────────────────────┐
│  1. 미니게임 선택/참여                                   │
│         ↓                                               │
│  2. 짧은 세션 플레이 (30초~1분)                          │
│         ↓                                               │
│  3. 스코어/보상 획득                                     │
│         ↓                                               │
│  4. 몬스터/파티 성장 ─────────────────────→ (1로 반복)    │
└─────────────────────────────────────────────────────────┘
```

### 루프 상세 설명

1. **미니게임 선택**: 다양한 미니게임 중 선택
2. **짧은 세션**: 30초~1분 내 완료되는 빠른 게임플레이
3. **보상 획득**: 스코어 기반 보상, 랭킹 보상
4. **성장**: 몬스터 수집/강화, 파티 레벨업

---

## 3. 재미 축 (Fun Pillars)

### 3.1 짧은 세션 (Short Sessions)
- **목표**: 짧은 시간에 강렬한 재미
- **핵심 경험**: 1분 만에 끝나는 시원한 플레이
- **주요 요소**:
  - 30초~1분 미니게임
  - 빠른 피드백
  - 즉각적 보상
  - 연속 플레이 유도

### 3.2 다양한 규칙 (Variety)
- **목표**: 매번 다른 게임을 하는 듯한 신선함
- **핵심 경험**: 이번엔 뭐 하지? 하는 기대감
- **주요 요소**:
  - 다양한 미니게임 풀
  - 일일 추천 게임
  - 시즌 한정 게임
  - 장르 믹스

### 3.3 성장/파티 (Growth/Party)
- **목표**: 미니게임 성과가 성장으로 연결
- **핵심 경험**: 미니게임 잘해서 몬스터가 강해졌다
- **주요 요소**:
  - 몬스터 수집
  - 파티 레벨
  - 누적 보상
  - 도감 시스템

## 4. 디자인 레버 (Design Levers)

### 4.1 초단타 미니게임
| 레버 | 설명 |
|------|------|
| 30초~1분 게임 | 30초~1분 내 끝나는 미니게임 여러 개 디자인 |
| 구현 방향 | 탭 게임, 피하기 게임, 매칭 게임, 타이밍 게임 등 10종 이상 |

### 4.2 스코어-성장 연결
| 레버 | 설명 |
|------|------|
| 미니게임 → 보상 → 성장 | 미니게임마다 스코어/보상/성장 연결 |
| 구현 방향 | 하이스코어 → 경험치 보너스, 연속 플레이 → 콤보 보상 |

### 4.3 재사용 가능 포맷
| 레버 | 설명 |
|------|------|
| 다른 게임 이벤트용 | 다른 게임 이벤트에 재사용 가능한 미니게임 포맷 정의 |
| 구현 방향 | 스킨 교체 가능 구조, API 연동, 보상 커스터마이징 |

## 5. 재미 측정 지표 (KPI)

| 지표 | 목표 | 측정 방법 |
|------|------|-----------|
| 일일 미니게임 횟수 | 20회 이상 | 게임 플레이 로그 |
| 평균 세션 길이 | 45초 | 게임 시작 → 종료 |
| 게임 다양성 | 일 5종 이상 | 플레이한 게임 종류 |
| 몬스터 수집률 | 월 50% 이상 | 도감 달성률 |
| D7 리텐션 | 35% 이상 | 7일차 잔존율 |
