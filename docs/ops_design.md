## 3. 1년차 라이브 플랜

### 3.1 초기 (1~3개월차)

| 주차 | 주요 활동 |
|------|----------|
| 1~2주 | 소프트 런칭, 버그 수정 |
| 3~6주 | 기본 미니게임 세트 밸런스 |
| 7~10주 | 몬스터 수집 시스템 안정화 |
| 11~12주 | 첫 이벤트 파티 준비 |

**핵심 과제:**
- 미니게임 다양성 확보
- 몬스터 수집 재미
- 짧은 세션 최적화

| 월 | 주요 업데이트 |

| 5개월 | 타이틀별 특별 룰/버프 도입 |
| 6개월 | 크로스 게임 연계 이벤트 |
| 7개월 | 신규 미니게임 추가 |
| 8개월 | 여름 파티 페스티벌 |

**핵심 과제:**
- 미니게임 로테이션 사이클
- 크로스 게임 연계 효과 검증
- 신규 미니게임 개발 사이클

### 3.3 후기 (9~12개월차)

| 월 | 주요 업데이트 |
|----|-------------|
| 9개월 | 0024 이벤트 연계 준비 |
