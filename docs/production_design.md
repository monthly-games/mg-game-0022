## 1. 재미 요소 (Fun Factors)

### 1.1 핵심 재미 (Core Fun)
| 요소 | 설명 | 강도 |
|------|------|------|
| **미니게임 다양성** | 30초~2분의 간단하고 다양한 미니게임 | ★★★★★ |
| **몬스터 육성** | 미니게임 보상으로 몬스터 성장 | ★★★★★ |
| **파티 플레이** | 친구들과 함께하는 협동/경쟁 | ★★★★☆ |
| **방치 성장** | 오프라인 시간 동안 자동 성장 | ★★★☆☆ |

### 1.2 만족감 요소 (Satisfaction)
- **퍼펙트 클리어**: 미니게임 별 3개 획득
- **몬스터 진화**: 진화 시 새로운 외형과 능력
- **파티 승리**: 친구들과의 협동 성공
- **새 게임 해금**: 다양한 미니게임 해금

### 1.3 도전 요소 (Challenge)
```
도전 구조:
├── 미니게임 난이도 (Easy~Extreme)
├── 별 3개 달성
├── PvP 토너먼트
├── 협동 클리어
└── 라이브 이벤트 게임
```

### 1.4 사회적 요소 (Social)
- **파티 모드**: 친구와 협동/대결
- **릴레이**: 순서대로 미니게임 릴레이
- **토너먼트**: 16인 토너먼트
- **리더보드**: 게임별/시즌별 랭킹

## 2. 게임 루프 (Game Loop)

### 2.1 코어 루프 (30초~2분) - 미니게임
```
┌─────────────────────────────────────────────────┐
│                    코어 루프                      │
├─────────────────────────────────────────────────┤
│  [게임 선택] → [미니게임 플레이] → [점수 계산]    │
│       ↑                            ↓             │
│       └──── [몬스터 경험치] ← [보상 획득] ←┘     │
└─────────────────────────────────────────────────┘
```

**단계별 상세:**
1. **게임 선택** (10초): 플레이할 미니게임 선택
2. **미니게임 플레이** (30초~2분): 게임 진행
3. **점수 계산** (3초): 별 등급 결정
4. **보상 획득** (5초): 코인, 경험치 획득
5. **몬스터 경험치** (즉시): 성장 반영

### 2.2 세션 루프 (5~15분)
```
[접속] → [방치 보상] → [미니게임] → [몬스터 레벨업] → [진화 확인]
                            ↓
                  [파티 플레이] ← [친구 초대]
```

### 2.3 메타 루프 (장기)
```
┌──────────────────────────────────────────────────────┐
│                     메타 진행                         │
├──────────────────────────────────────────────────────┤
│  [미니게임 플레이] → [코인/경험치] → [몬스터 성장]    │
│         ↓                              ↓             │
│  [새 게임 해금]                  [몬스터 진화]        │
│         ↓                              ↓             │
│  [시즌 이벤트] ←───────────────────────┘             │
└──────────────────────────────────────────────────────┘
```

### 2.4 플로우 다이어그램
```
[앱 시작]
    ↓
[방치 보상]
    ↓
[메인 화면] ─────→ [몬스터 관리]
    │              [미니게임 목록]
    │              [상점]
    │              [설정]
    ↓
[솔로] → [게임 선택] → [플레이] → [결과]
    │
    └── [파티] → [친구 매칭] → [협동/대결] → [랭킹]
```

## 4. 레벨 디자인 (Level Design)

### 4.1 난이도 곡선
```
난이도
  ▲
  │                              ╭── Extreme
  │                         ╭────╯ Hard
  │                    ╭────╯ Normal
  │               ╭────╯ Easy
  │          ╭────╯
  │     ╭────╯
  │╭────╯
  └──────────────────────────────────────→ 몬스터 레벨
    1    20   50   100  150  200  300
```

### 4.2 콘텐츠 진행
| 레벨 | 해금 요소 | 미니게임 카테고리 |
|------|----------|-------------------|
| 1~20 | 시작 | 반사신경 기본 게임 |
| 21~50 | Lv.20 | 퍼즐, 기억력 게임 |
| 51~100 | Lv.50 | 전략, 리듬 게임 |
| 101~150 | Lv.100 | 고급 난이도, 파티 모드 |
| 151~200 | Lv.150 | 익스트림 난이도 |
| 201~300 | Lv.200 | 전설 몬스터 전용 게임 |

### 4.3 밸런싱 공식
```dart
// 미니게임 점수
int calculateScore(MiniGame game, Performance perf) {
  int baseScore = game.maxScore;
  double accuracy = perf.correctActions / perf.totalActions;
  double timeBonus = max(0, 1 - perf.time / game.timeLimit) * 0.2;
  return (baseScore * accuracy * (1 + timeBonus)).toInt();
}

// 별 등급
int calculateStars(int score, MiniGame game) {
  if (score >= game.perfectThreshold) return 3;
  if (score >= game.goodThreshold) return 2;
  if (score >= game.passThreshold) return 1;
  return 0;
}

// 몬스터 경험치
int expGained(int stars, int difficulty, Monster monster) {
  int base = 10 * stars * difficulty;
  double typeBonus = monster.isActive ? 1.5 : 1.0;
  return (base * typeBonus).toInt();
}

// 방치 수익
int idleReward(Monster monster, Duration offline) {
  int perHour = monster.level * 10;
  int maxHours = 24;
  return min(offline.inHours, maxHours) * perHour;
}
```

### 4.4 보상 체계
| 보상 유형 | 획득 조건 | 수량 |
|-----------|----------|------|
| 코인 | 게임 클리어 | 별 기반 50~300 |
| 경험치 | 게임 클리어 | 난이도별 10~100 |
| 다이아 | 퍼펙트, 업적 | 1~50 |
| 몬스터 조각 | 가챠, 이벤트 | 1~10 |
| 방치 보상 | 시간 경과 | 시간당 |

## 5. UI/UX 설계

### 5.1 정보 계층
```
레벨 1 (항상 표시):
├── 코인/다이아
├── 티켓
├── 현재 몬스터
└── 이벤트 배너

레벨 2 (상호작용 시):
├── 게임 선택 화면
├── 몬스터 상세
├── 파티 로비
└── 리더보드

레벨 3 (팝업):
├── 게임 결과
├── 몬스터 레벨업
├── 진화 연출
└── 가챠 결과
```

### 5.2 네비게이션 흐름
```
[메인 화면]
    ├── [미니게임] → [카테고리] → [게임 선택] → [플레이]
    ├── [몬스터] → [목록] → [강화/진화]
    ├── [파티] → [친구/매칭] → [협동/대결]
    ├── [가챠] → [뽑기]
    ├── [상점] → [상품 목록]
    └── [설정] → [옵션]
```

### 5.3 피드백 시스템
| 이벤트 | 시각 | 청각 | 촉각 |
|--------|------|------|------|
| 정답 | 성공 이펙트 | 정답음 | - |
| 오답 | 실패 이펙트 | 오답음 | 진동 (약) |
| 별 3개 | 퍼펙트 연출 | 팡파레 | 진동 (중) |
| 레벨업 | 레벨업 연출 | 레벨업음 | 진동 (약) |
| 진화 | 진화 연출 | 진화 사운드 | 진동 (강) |

### 5.4 접근성
- **간단한 조작**: 탭 기반 미니게임
- **난이도 선택**: 4단계 난이도
- **색맹 모드**: 아이콘/패턴 추가
- **폰트 크기**: 3단계 조절
- **자동 플레이**: 방치 모드
